# JavaScript Basics

## What is JavaScript?
JavaScript is the language that makes websites interactive! You can use it to create games, buttons, and animations.

## Adding JavaScript
1. **Inline**
```html
<button onclick="alert('Hello!')">Click Me</button>
```

2. **Internal**
```html
<script>
  alert("Page loaded!");
</script>
```

3. **External**
```html
<script src="script.js"></script>
```

## Basic Concepts

### Variables
```javascript
let name = "Alice";
```

### Functions
```javascript
function sayHello() {
  alert("Hello!");
}
```

### Events
```javascript
document.querySelector("button").addEventListener("click", sayHello);
```
